# 代码框

这个比较简单，只需要用反引号“\`”即可

单行代码框可以用“ \`code\` ”来实现

多行代码框可以用“\```code \```”实现

例:

`print("HelloWorld!")`

``` 在这里可以写代码注释
#include <iostream>
using namespace std;
int main(){
  cout << "HelloWorld!" << endl;
  return 0;
}
```

# 表格

表格用“|”和“-”来划定行列

":----"表示靠左

"----:"表示靠右

":---:"表示居中
