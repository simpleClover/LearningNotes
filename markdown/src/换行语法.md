# 换行语法

在一行的末尾输入两个及以上的空格，然后回车，便可创建一个换行

下一例子将用下划线`_`表示末尾空格
```
这是第一句话。__
这是第二句话，与上一句话相连但是是新的一行。

这是第三句话，与上不相连所以是新的段落。
```
结果：
```
这是第一句话。
这是第二句话，与上一句话相连但是是新的一行。

这是第三句话，与上不相连所以是新的段落。
```

虽然几乎所有markdown应用程序都支持句末空格换行，但是由于在编辑器中难以直接看到空格，因此还有另外一种换行方式：使用HTML的`<br>`标签

```
这是第一句话。<br>
这是第二句话，与上一句话相连但是是新的一行。

这是第三句话，与上不相连所以是新的段落。
```
结果：
```
这是第一句话。
这是第二句话，与上一句话相连但是是新的一行。

这是第三句话，与上不相连所以是新的段落。
```

另外，还有几种轻量级标记语言支持在行末添加`\`换行，但是为了兼容性，还是更推荐使用`结尾空格`和`<br>`两种方式换行
